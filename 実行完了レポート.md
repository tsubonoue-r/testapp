## 🌐 アクセス情報

### URL

- **メインアプリ**: http://localhost:3000
- **カメラ撮影**: http://localhost:3000/camera.html
- **静的ファイル**: http://localhost:3000/index.html
- **アップロード写真**: http://localhost:3000/uploads/

### API エンドポイント

#### 認証
- `POST /api/auth/register` - ユーザー登録
- `POST /api/auth/login` - ログイン
- `GET /api/auth/me` - 現在のユーザー情報

#### 案件管理
- `GET /api/projects` - 案件一覧
- `POST /api/projects` - 案件作成
- `GET /api/projects/:id` - 案件取得
- `PUT /api/projects/:id` - 案件更新
- `DELETE /api/projects/:id` - 案件削除
- `GET /api/projects/search/query?q=キーワード` - 検索

#### 工事看板
- `GET /api/signboards` - 工事看板一覧
- `POST /api/signboards` - 工事看板作成
- `GET /api/signboards/:id` - 工事看板取得

#### 写真管理
- `GET /api/photos` - 写真一覧
- `POST /api/photos/upload` - 写真アップロード
- `GET /api/photos/:id` - 写真取得
- `PUT /api/photos/:id` - 写真更新
- `DELETE /api/photos/:id` - 写真削除

## 📱 スマホでの使用方法

### カメラ撮影手順

1. スマホで http://localhost:3000/camera.html にアクセス
2. カメラへのアクセス許可を承認
3. 案件を選択
4. 撮影ボタンをタップ
5. プレビューを確認
6. 「アップロード」をタップ

### メインアプリ使用手順

1. http://localhost:3000 にアクセス
2. タブで「案件」「看板」「写真」を切り替え
3. 各セクションで作成・編集が可能
4. FABボタン(+)で新規作成
